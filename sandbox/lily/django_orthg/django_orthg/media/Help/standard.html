The linear least squares problem is: 

                    \min_{x}\left \| b - Ax \right \| _{2},
where A an m-by-n matrix and b is a given m element vector and x is the solution. If $m \ge n$ and B$\mbox{rank}(A) = n$, the solution to problem is unique, and the problem is also referred to as finding a least squares solution to an overdetermined system of linear equations.
