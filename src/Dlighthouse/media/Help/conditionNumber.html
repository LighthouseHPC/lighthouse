The condition number of a system of linear equations Ax=b measures the stability to numerical operations. A system of equations is considered to be well-conditioned/ill-conditioned if a slight change in the coefficient matrix A or in the vector b results in a small/great change in the solution vector x.

The condition number of a square matrix A is defined as: 

                cond(A) = ||A||||A-1||,
where ||A|| is the norm of matrix A.

The condition estimation routines in LAPACK compute the reciprocal condition number, RCOND, which is defined as: 

                RCOND = 1/cond(A).

Condition numbers near one indicate well-conditioned matrices. Condition numbers much greater than one imply ill-conditioned matrices.
